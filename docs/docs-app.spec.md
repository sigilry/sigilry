# Docs App Sub-Spec

## Overview

The docs app (`@sigilry/docs`) builds and publishes developer documentation for the monorepo.

## Scope

- Build Astro/Starlight documentation content.
- Publish API reference artifacts and site navigation.
- Support local docs development and preview workflows.

## Key Constraints

- Docs structure and generated API references should stay aligned with package exports.
- Pages should remain AI-friendly (structured headings, copy-paste code, stable links).

## Canonical Spec

The full documentation site spec lives in the top-level `SPEC.md`.
